Finquanta AI Server API

Comprehensive financial management API documentation

🚀 Quick Start

Base URL: http://localhost:3001/api/v1

Authentication: Bearer Token (JWT)

API Version: 1.0.0

🔐 Authentication

To use authenticated endpoints:

  1. Register a new account: POST /auth/register
  2. Login to get tokens: POST /auth/login
  3. Include the access token in the Authorization header: Authorization: Bearer <your-token>

Note: Access tokens expire after 15 minutes. Use the refresh token to get new access tokens.

📝 Example Requests

1. Register User

curl -X POST "http://localhost:3001/api/v1/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john.doe@example.com",
    "password": "SecurePass123!",
    "firstName": "John",
    "lastName": "Doe"
  }'

2. Login

curl -X POST "http://localhost:3001/api/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john.doe@example.com",
    "password": "SecurePass123!"
  }'

3. Create Transaction

curl -X POST "http://localhost:3001/api/v1/financial/transactions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -d '{
    "type": "income",
    "category": "Salary",
    "amount": 5000.00,
    "description": "Monthly salary",
    "date": "2024-01-15"
  }'

4. Get Transactions

curl -X GET "http://localhost:3001/api/v1/financial/transactions?limit=10&type=income" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
